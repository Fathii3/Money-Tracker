##
#### Deskripsi Singkat Aplikasi

Aplikasi ini dirancang untuk memudahkan pengguna dalam:

1. Mengelola langganan bulanan
2. Mencatat pemasukan dan pengeluaran
3. Melihat ringkasan keuangan secara visual

4. Mengontrol anggaran agar pengeluaran tetap terjaga
5. Dengan antarmuka yang sederhana dan intuitif, aplikasi ini cocok digunakan untuk kebutuhan pengelolaan keuangan sehari-hari.

##
#### Fitur pada Aplikasi
- Dashboard Keuangan Menampilkan ringkasan kondisi keuangan dan langganan aktif secara visual.

- Manajemen Langganan Mencatat dan memantau langganan bulanan seperti streaming dan layanan digital.

- Pemasukan & Pengeluaran Pencatatan transaksi keuangan secara terpisah dan terstruktur.

- Kalender Transaksi Melihat histori keuangan berdasarkan tanggal.

- Monitoring Anggaran Membantu pengguna mengontrol pengeluaran agar tidak melebihi batas.

- Pengaturan Aplikasi Preferensi tema dan pengaturan aplikasi lainnya.


##
####  Stack Teknologi 

**1. Flutter Version:** Flutter 3.7.12

**2. Android Version:** Android 16

**3. UI Framework:** Material Design

**4. Local Storage:** Shared Preferences


**5. Chart & Visualization:** fl_chart

**6. Calendar & Date Handling:** table_calendar

**7. Localization:** intl, flutter_localizations

**8. Custom Local Packages:** calendar_agenda, card_swiper

##
#### Cara Menjalankan Aplikasi

- Buka GitHub Repository proyek ini

- Masuk ke menu Releases

- Unduh file APK terbaru (app-release.apk)

- Pindahkan APK ke perangkat Android (jika perlu)

- Aktifkan Install from Unknown Sources
